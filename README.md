# quote_convert

Конвертирование биржевых котировок из тиковых данных в range-бары с сохранением в SQLite.

## Описание

Инструмент для преобразования тиковых данных фьючерса RTS (из CSV/ZIP архивов) в range-бары заданного размера. Range-бары формируются по ценовому диапазону (а не по времени), что позволяет фильтровать рыночный шум.

## Структура проекта

```
├── convert_to_range/
│   ├── rts_tick_zip_csv_to_db.py    # Основной конвертер: тики → range-бары → SQLite
│   ├── first_record_day_db.py       # Выбор файла с самой ранней датой
│   └── sqlighter3.py                # Модуль работы с SQLite (создание таблиц, вставка)
```

## Принцип работы

1. Читает тиковые данные из CSV-файлов (ZIP-архивы)
2. Формирует range-бары заданного размера (в пунктах)
3. Записывает результат в SQLite базу данных

## Зависимости

- Python 3.10+
- pandas, numpy
